**Venda de Ingresso de Teatro**
---
Sistema de venda de ingresso para teatro. 

Funcionalidades
---
- Inserção de CPF
- Escolha de turno
- Tipos de ingresso (Plateia A, Plateia B, Friza e Nobreza)
- Cálculo do preço total
- Geração e impressão de comprovante

Tecnologias
---
- HTML
- CSS
- JavaScript
-  (banco de dados)
-  (autenticação)
